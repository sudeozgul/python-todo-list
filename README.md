# Python To-Do List Application

This project is a simple command-line based To-Do List application developed using Python.

## Features
- Add new tasks
- Prevent duplicate tasks
- Delete tasks
- Save tasks to a text file
- Load tasks from file on startup

## Technologies
- Python
- File handling (read/write)
- Basic error handling

## Purpose
This project was developed during an internship process to gain hands-on experience with Python programming and file operations.
